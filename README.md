# Maturitka 2026 - KBB

Toto repo je určené pro studenty SSPŠ (smíchovského areálu) pro přípravu na maturitní zkoušky pro rok 2026 v oboru KBB. Kontribuce jsou vítány


## Konvence

### Otázky (KBB, PVA)

Každá otázka je minimálně jeden soubor. Ostatní soubory k otázkám dávejte do `<KBB nebo PVA>/assets/` nebo vlastního adresáře pro otázku.

Všechny dokumenty doporučuji psát ve formátu [markdown](https://www.markdownguide.org/), ze kterého se pak dají jednoduše převézt do `.docx` nebo `.pdf`.

Soubory s otázkami mají jména v následujícím formátu: `<číslo otázky> - <jméno otázky>.md`
- Např. `1 - první otázka.md`
